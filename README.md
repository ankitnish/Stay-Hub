# Airbnb Clone – Full Stack Web Application

## Overview

Airbnb Clone is a full-stack web application inspired by Airbnb that enables users to discover, list, and book accommodations. The platform provides a seamless experience for property owners to manage listings and for travelers to explore and reserve stays.

## Features

* User Authentication & Authorization (Sign Up, Login, Logout)
* Create, Edit, and Delete Property Listings
* Upload Property Images
* Search and Browse Listings
* Detailed Property Information Page
* Booking and Reservation System
* Responsive User Interface
* Secure Session Management
* User Reviews and Ratings
* Interactive Maps Integration

## Project Architecture

The application follows the MVC (Model-View-Controller) architecture to maintain clean code organization and scalability.
